SHABR

The purpose of this project was to see if I could use some of the new bootstrap skills to recreate the example website that we were given in our unique way.I had to use the bootstrap website to learn new tools that I could use to recreate this example website in my own way.I was given all the information that had to be put into the website and it was up to me to design it using bootstrap. Additionally I had to use the grid system which helped make my website more organized knowing where each class is.This was my first creating a whole website using bootstrap. Creating this website was fun but challenging at the same time. If I faced a problem I always had the Bootstrap website and my classmates on slack for help.
